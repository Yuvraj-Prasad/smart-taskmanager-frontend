Task Manager Frontend

This is the frontend part of my Task Manager application. It is built using React Native and connects to my deployed backend API for authentication and task management.

Live Backend API

The frontend is connected to this backend:
https://smart-taskmanager-backend-1.onrender.com

Tech Stack Used

React Native
Expo
Axios
React Navigation
Context API

Features

User Registration
User Login
JWT Authentication
Create Task
Update Task
Delete Task
Filter Tasks by Status
Search Tasks

How to Run Locally

First install dependencies:
npm install
Then start the app:
npm start
If using Expo:
npx expo start
You can then run it on an Android emulator, iOS simulator, or open it in a browser.

API Configuration

The frontend uses Axios with the deployed backend URL:
https://smart-taskmanager-backend-1.onrender.com
